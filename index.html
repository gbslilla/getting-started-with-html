People use browsers and search engines to make requests through the internet. These requests exist in a form of HTTP which is protocol the browser uses to make these requests. HTTP requests are accepted by the servers which are computers that hosts data and sends files back in reply to requests.

HTML or Hypertext Markup Language is the most common type of contents you can find on the web together with pictures, music, links and other documents. HTML is a coding language that creates websites. It consists of links, tags or markups and the language code itself.


HTML mark ups

Mark ups or tags are functions that give order to the your computer.
HTML is made up of elements. Each element consist of an

- opening tag,
- contents,
- closing tag.

< opening tag > data < / closing tag >

example:
This word is bold.

The tags gave order to my computer to show the word bold.


Tags

< b > contents < /b > - b stands for bold and make the word bold.
< em > contents < /em> - em stands for emphasized.
< i > contents < /i > - i stands for italic and make the contents slanted.


Attributes

You can also include links in HTML contents and the way to do it is to include a:

- tag with an attribute and the particular value,
- a word, part of your text or your "link"
- and a closing tag.

< tag ATTR = "value" > contents < closing tag >
< a href = "URL" > your chosen word < /a >

< a > stands for anchor,this is the opening tag,
href is the name of the attribure,
< /a > is the closing tag

example:
This website is your best friend.

Some tags does not have contents, these tags are called void tags. Image is one of these void tags.

Tags are also distinguished by their property of being a part of an inline or block element. Inline tags can be placed into the text-line without breaking that while block elements creat an invisible block around the text breaking the structure. These block elements can also be altered by other coding languages as CSS or JS.

Inline tags

< a > anchor,
< b > bold,
< br > break,
< em > emphasized,
< i > italic,
< img > image,
< span > Span makes it possible to alter the word's size, colour by other coding language.

Block tags

< div > Similar to span but a block tag,
< form >
